Check print_sorted() method

Initialize MyList and append elements
>>> MyList = __import__('1-my_list').MyList
>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)
>>> print(my_list)
[1, 4, 2, 3, 5]

Check print_sorted() sorts the list without modifying original list
>>> my_list.print_sorted()
>>> print(my_list)
[1, 4, 2, 3, 5]
