Check initialization and basic append functionality

>>> MyList = __import__('1-my_list').MyList
>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)
>>> print(my_list)
[1, 4, 2, 3, 5]

Check print_sorted() method sorts the list in ascending order

>>> my_list.print_sorted()
>>> print(my_list)
[1, 2, 3, 4, 5]

Check print_sorted() does not modify the original list

>>> my_list.print_sorted()
>>> print(my_list)
[1, 2, 3, 4, 5]

Check sorting with duplicate elements

>>> my_list.append(2)
>>> my_list.append(1)
>>> my_list.print_sorted()
>>> print(my_list)
[1, 1, 2, 2, 3, 4, 5]

Check sorting with negative numbers

>>> my_list.append(-5)
>>> my_list.append(-2)
>>> my_list.print_sorted()
>>> print(my_list)
[-5, -2, 1, 1, 2, 2, 3, 4, 5]

Check sorting with a large list

>>> large_list = MyList()
>>> for i in range(1000, 0, -1):
...     large_list.append(i)
>>> large_list.print_sorted()
>>> print(large_list)
[1, 2, 3, ..., 1000]
